# Climate Against Conflict

A repository dedicated to analyzing and addressing the intersection of climate change and conflict in South Sudan and Sub-Saharan Africa in general. This project explores how environmental stressors, resource scarcity, and climate impacts contribute to global conflicts, and develops tools and resources to promote climate resilience and peace. The QGIS Software will also be used to create a variety of visualizations as well.

## Features

- Climate and conflict data analysis
- Environmental impact assessments
- Geospatial mapping and visualization
- Policy recommendations
- Case study documentation
